# 📖 私有雲端日記帳 (Private Cloud Expense Tracker)

歡迎來到「私有雲端日記帳」！這是一款**完全免費、無廣告**，且**資料 100% 掌握在自己手裡**的輕量級開源記帳 Web App。

有別於市面上的記帳軟體將您的財務隱私存在別人的伺服器，本系統的資料庫就是**您專屬的 Google 雲端硬碟 (Google Sheets)**。您的財務資料，只有您自己看得到！

## ✨ 系統特色
* **資料隱私絕對安全**：後端資料庫建立在使用者的 Google 帳號下，開發者無法碰到任何一筆資料。
* **無縫跨平台體驗**：支援 PWA (Progressive Web App)，可直接將網頁加入手機桌面，體驗如原生 App 般流暢。
* **輕量無負擔**：前端託管於 GitHub Pages，後端使用 Google Apps Script (GAS) 驅動，零伺服器成本。

## 🚀 快速安裝與使用手冊 (3 分鐘搞定)

只要跟著以下 3 個簡單的步驟，就能打造專屬於您的記帳系統。

### 步驟 1：一鍵複製專屬資料庫 (後端)
1. 請點擊這個連結：https://docs.google.com/spreadsheets/d/1gf9ZudCgBN3u5c_-jYL6OpMjrJabZIM8j1ApbfBmOLY/copy
2. 畫面會直接跳出提示，請點擊畫面上的 **「建立副本」** 按鈕。
3. 恭喜！現在您的 Google 雲端硬碟裡已經建立好專屬的記帳資料庫了。您可以點擊左上角，隨意更改這份試算表的名稱。

### 步驟 2：發布並取得「API 網址」 (⭐ 最重要的一步)
為了讓記帳 App 可以把資料寫入您的試算表，我們需要產生一串專屬的網址。
1. 在剛剛複製的試算表中，點擊上方選單的 **「擴充功能」** ➔ **「Apps Script」**。
2. 畫面會開啟程式碼編輯器，請直接點擊右上角的 **「部署」** ➔ **「新增部署作業」**。
3. 在彈出視窗中，點擊左上角的「齒輪 ⚙️」圖示，選擇 **「網頁應用程式」**。
4. 設定以下選項（**選錯會無法連線喔！**）：
   * **執行身分：** 選擇 `我` (您的 Gmail)。
   * **誰可以存取：** 選擇 `所有人` (Anyone)。
5. 點擊右下角的 **「部署」**。
   * *(注意：Google 為了保護安全，會跳出需要授權的警告。請點擊「審查權限」➔ 選擇帳號 ➔ 點擊左下角「進階」➔ 點擊前往➔ 點擊「允許」。)*
6. 部署完成後，畫面會出現一串 **「網頁應用程式網址」**，請點擊旁邊的 **「複製」** 按鈕把這串網址記下來！

### 步驟 3：連結您的記帳 App (前端)
1. 開啟記帳 App 網址：https://famidoc.github.io/day-book/
2. 第一次開啟時，畫面會跳出「**連結您的雲端資料庫**」的視窗。
3. 把剛剛在步驟 2 複製的網址貼進去，點擊 **「儲存並開始使用」**。
4. 🎉 系統連線成功！現在就把它「加入手機主畫面」開始記帳吧！

> **💡 溫馨提示：** 未來如果您想換一個試算表，或者網址貼錯了，只要點擊 App 右上角的「⚙️ 齒輪」圖示，就可以隨時重新設定。
